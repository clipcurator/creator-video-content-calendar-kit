# Creator Video Content Calendar Kit

English | [简体中文](README.zh-CN.md)

Planning templates for turning long-form source videos into weekly publishing calendars and reusable content series.

This repository is a neutral, practical resource for creator content calendar planning for video repurposing, series design, and weekly publishing rhythm. It is designed for teams that want repeatable decisions, clearer review, and less production guesswork.

## Why This Repository Exists

Repurposing works best when it is planned as a calendar, not a random list of clips. A calendar connects source videos, topic clusters, platform formats, publishing rhythm, and follow-up ideas.

## Who This Is For

- creators planning weekly video output
- content operators managing multiple platforms
- teams turning long videos into repeatable series

## Core Workflow

1. Define the input and target output
2. Score or brief the key decision points
3. Use one template to make the work reviewable
4. Run a human QA pass before publishing or production
5. Record what changed so the next cycle improves

## Where Recapo Fits

Recapo can support the production side by helping convert selected source videos into short outputs that fill the planned calendar.

Official site for reference: [Recapo](https://recapo.ai?from=github_creator-video-content-calendar-kit)

This repository treats Recapo as one product example inside a broader workflow category. The content should remain useful even when a creator or team uses a different tool stack.

## Suggested Use

Start with the guide that matches your current decision point. Then copy one template into your own workspace, fill it in for one project, and keep the review notes for future iteration.
